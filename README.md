## Task Overview
A basic keylogger that records and logs every keystroke
to a text file. Handles both regular and special keys
with timestamps for each session.

## Features
- Records all keystrokes in real time
- Handles special keys like Space, Enter, Backspace
- Requires user confirmation before starting
- Timestamps each session in the log file
- Press Escape to stop safely

## Tech Stack
- Language: Python 3.x
- Library: pynput
- Concepts: Event listeners, File I/O, Keyboard monitoring

## How to Run
pip install pynput
python keylogger.py
